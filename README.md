# Hangwoman CMS


### Install it and run:

```sh
yarn
yarn dev
```

### Run tests

```sh
yarn test:jest
```

## Deploying to Heroku git

Login to Heroku

```
heroku login
```


Link repository

```
git remote set-url heroku https://git.heroku.com/hangwoman-cms.git
```


Publish changes

```
git push heroku master
```
